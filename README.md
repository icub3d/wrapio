wrapio
======

[![Build Status](https://drone.io/github.com/icub3d/wrapio/status.png)](https://drone.io/github.com/icub3d/wrapio/latest)

interfaces.

You can find documentation on using the package at
http://godoc.org/github.com/icub3d/wrapio.
